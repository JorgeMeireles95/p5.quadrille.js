Creates a filled quadrille from a 2D array which may contain any combination of zeros (for empty cells), p5 colors and chars, including emojis.

linkSyntax

createQuadrille(array2D)

createQuadrille(width, height)

linkParameters

array2D2D array which may contain any combination of zeros (for empty cells), p5 colors and chars, including emojis.
widthnumber of columns
heightnumber of rows

Creates an initiallly empty quadrille, a board, having width * height cells. See the examples.

linkdrawQuadrille(quadrille, row, col)

Draws the quadrille at (row, col). See the examples.
