# `clear()`

Sets quadrille cells to `0`. Either all quadrille cells or a given `row`.

# Syntax

> `clear()`

> `clear(row)`

# Parameters

| <!-- --> | <!-- -->                                                                    |
|----------|-----------------------------------------------------------------------------|
| row      | Number: number of the row to be cleared [\[0..height\]](/docs/props#height) |
