# `clear()`

Sets quadrille [memory2D](/docs/props#memory2d) entries to `0`. Either all quadrille entries or a given `row`.

# Syntax

> `clear()`

> `clear(row)`

# Parameters

| <!-- --> | <!-- -->                                                                    |
|----------|-----------------------------------------------------------------------------|
| row      | Number: number of the row to be cleared [\[0..height\]](/docs/props#height) |
