# `magnitude()`

Returns the number of non-empty cells in the given row of the quadrille [memory2D](/docs/props#memory2d) internal representation.

# Syntax

> `magnitude(row)`

# Parameters

| <!-- --> | <!-- -->                                                  |
|----------|-----------------------------------------------------------|
| row      | Number: number of the row [0..[width](/docs/props#width)] |
