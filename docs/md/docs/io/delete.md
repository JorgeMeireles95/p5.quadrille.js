# `delete()`

Deletes a row from the quadrille [memory2D](/docs/props#memory2d) internal representation.

# Syntax

> `delete(row)`

# Parameters

| <!-- --> | <!-- -->                                                                    |
|----------|-----------------------------------------------------------------------------|
| row      | Number: number of the row to be deleted [\[0..height\]](/docs/props#height) |
